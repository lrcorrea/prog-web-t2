Observações sobre o UIKit:

O UIKit é bem rico em components prontos para serem utilziados como: carousel, modal, inputs, tooltip, etc. Para utilizar é muito simples, basta importar o js e o css.
Possui propiedades modernas do css3 e dando suporte nos navegadores em geral contando com o IE 11. Conta também com as propiedades grid e flexbox facilitando a organização dos conteúdos.
Um ponto negativo que achei foi as opções de cores que ele disponibiliza, mas é possível adicionar arquivos sass ou less e adicionar cores ou mudar e até mesmo mudar os valores padrões como os breakpoints
Possui uma biblioteca de ícones, mas precisa importar separado do framework principal. Ela possui os princípais ícones mas não tem tantos disponíveis.
As imagens possuem "lazyload" para melhorar o desempenho dos sites e várias opções de animação para usar.
A framework é bem completa com muitos components prontas e é possível fazer um site bonito e com firulas

O framework possui facil utilizaçao, possui um codigo simples e limpo. Para sites que possuem muitos componentes ele acaba sendo simples demais.
A documentaçao dele é bem completa e torna facil a aplicaçao do mesmo.
O framework se encaixa muito bem para desenvolvimento rapido de sites ou até mesmo para a criaçao de prototipos. A documentaçao original do framework é bem explicada e de facil utilizaçao. Mesmo tendo um estilo mais minimalista o UiKit pode ser facilmente costumizavel.
É um framework muito bom para o desenvolvimento de prototipos ou de sites mais minimalistas. Existem outros frameworks que acabam sendo mais completos.
